# Alldle

It's like Wordle, but with ALL of the words.


## Word Lists

The full dictionary of guessable words comes from the complete [Common English Lexicon](https://github.com/Fj00/CEL) list, which contains ~70,000 of the most common english words. It does not include proper nouns.

Possible Answers use the "Top 5000 Lemmas" word list from [Word Frequency Data](https://www.wordfrequency.info)
based on the Corpus of Contemporary American English (COCA). Proper nouns as well as words
containing symbols or spaces were removed.
